# EasyTiwu · AI 练题工具

基于 Spring Cloud 微服务架构的智能练题平台，支持题库文件上传、在线练习和 AI 智能解析。

## 开发环境要求

- **Java 开发环境**: JDK 17
- **数据库环境**: MySQL 8
- **Node.js 运行环境**: 版本 22
