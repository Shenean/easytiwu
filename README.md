## 一、项目简介

EasyTiwu 是一个开源的智能刷题平台，旨在帮助用户将纸质或电子资料快速转换为线上私有题库，支持：

- 全量练习
- 分题型训练
- 错题集复盘
- 题库合并

通过这些功能，提升学习效率与效果。
